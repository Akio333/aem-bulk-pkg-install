# AEM Bulk Package Installer

A VS Code extension to upload, install, and backup multiple AEM packages or bundles right from the VS Code Explorer context menu.

## Features
- AEM: Upload file(s)
- AEM: Install file(s)
- AEM: Backup package(s)

## Configuration
Go to VS Code Settings and search for `Aem Bulk Installer`:
- `aemBulkInstaller.server.url`: AEM Server URL
- `aemBulkInstaller.server.port`: AEM Server Port
- `aemBulkInstaller.server.username`: Admin Username
- `aemBulkInstaller.server.password`: Admin Password
